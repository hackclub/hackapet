# Hridya's cat game

Pet game where you take care of your cat.

You send it to sleep / play with it / feed it / cure it from sickness.

Controls for port.py:
- Button 1: Feed
- Button 2: Cure
- Button 3: Play, long press for sleep

On main.py:
- 1: Feed
- 2: Cure
- 3: Play
- 4: Sleep

Hope you like it :D
